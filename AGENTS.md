# WorkFit Agents Context Document

## 1. 서비스 개요 (One-liner)
WorkFit은 이력서스펙 중심 채용을 대체하여, **실제 업무 문제 해결 능력으로 인재를 검증하는 실전형 검증 채용 플랫폼**이다.

기업은 채용 공고 대신 업무 문제를 제시하고, 지원자는 제한 시간 내 문제 해결안을 제출한다.  
AI는 제출된 해결안을 다각도로 평가하고, 기업은 점수와 핵심 피드백만을 기반으로 채용 결정을 내린다.

---

## 2. 문제 정의 (Problem)

기업이 인재를 채용하지 못하는 주요 원인은 다음과 같다.

- 기업은 **즉시 실무 투입 가능한 역량**을 원하지만 이를 사전에 검증할 방법은 이력서면접 외에 거의 없다.
- 수시직무 중심 채용이 확산되었지만 실제 업무 적합도를 판단할 수 있는 **정량적 기준이 부재**하다.
- 구직자는 과도한 스펙 경쟁과 실제 업무와 동떨어진 채용 과정으로 인해 **입사 후 괴리감**을 겪는다.
- 이는 **신입주니어 조기 이탈**의 주요 원인이 된다.

현재 채용은 문제를 해결할 수 있는 사람이 아니라 문제를 잘 포장한 사람을 선별하는 구조에 가깝다.

---

## 3. 타겟 사용자 (Target)

### 개인 사용자
- 취업 준비생
- 이직을 고려하는 직장인
- 스펙보다 **실제 실력으로 평가받고 싶은 사용자**

### 기업 사용자
- 수시 채용을 진행하는 기업
- 직무 역량 검증 비용을 줄이고 싶은 스타트업/중소기업
- 실무 중심 인재를 빠르게 선별하고 싶은 채용 담당자

---

## 4. 해결책 요약 (Solution)

### 전체 흐름
1. **기업**
   - 채용 공고 대신 직무 기반 문제를 작성
   - 문제에 포함될 핵심 평가 항목과 가중치를 설정
   - AI의 도움을 받아 공고(문제)를 완성
2. **지원자**
   - 공고를 확인하기 전까지 문제 내용은 블라인드
   - 지원과 동시에 문제 공개
   - 제한 시간 내 해결안 제출
3. **AI**
   - 제출된 해결안을 자동 분석
   - 항목별 점수 + 구체적인 피드백 생성
4. **기업**
   - 지원자의 **점수, 피드백 요약, 합불 여부**만 확인
   - 이력서스펙은 참고 자료 수준으로만 사용

---

## 5. 핵심 차별점 (Differentiation)

1. **스펙 비교가 아닌 문제 해결력 비교**
   - 학력, 연차, 자격증이 아닌 실제 결과물로 평가
2. **기업의 채용 리스크 감소**
   - 입사 전 실무 유사 문제를 통해 사전 검증
3. **구직자의 부담 감소**
   - 불필요한 스펙 경쟁 제거
   - 내가 할 수 있는 일로 평가받는 구조
